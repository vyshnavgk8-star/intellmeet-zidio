# IntellMeet

A collaborative meeting management platform built using MERN Stack.

## Features

- User Authentication
- JWT Security
- Meeting Management
- Participant Management
- Video Meetings (Jitsi)
- Meeting Chat
- Protected Routes

## Tech Stack

Frontend:

- React
- Axios
- React Router
- Bootstrap

Backend:

- Node.js
- Express.js
- MongoDB Atlas
- JWT
- bcrypt

## Installation

Backend:

```bash
cd backend
npm install
npm run dev
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```
